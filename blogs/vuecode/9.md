**lifecycle中写入调用生命周期的方法**  
如果$options配置项中存在该hook,则依次执行该hook中订阅/收集到的方法
``` js
//生命周期调用
export function callHook(vm,hook){
  const handlers = vm.$options[hook]
  if(handlers){
    for( let i=0;i<handlers.length;i++){
      handlers[i].call(this) //将全局vm指向当前实例的vm
    }
  }
}
```
